Техническое задание по ГОСТу 19
-----1. Введение
1.1 Наименование программы 
Мобильный кулинарный сайт: "Ваши рецепты".
1.2 Область применения
Сайт предназначен для поиска кулинарных рецептов. 
1.3 Объект, в котором используют программу
Сайт доступен для всех, кто интерерсуется кулинарией.
-----2. Назначение разработки
а. Пропаганда здоровой домашней пищи.
б. Показать преимущество домашней еды над фастфудом.
в. Повышение кулинарных навыков.
-----3. Требования к программе
3.1 Требования к функциональным характеристикам
Сайт имееет следующий функционал:
а. Выбор категории блюда
б. Просмотр пошагового рецепта блюда
-----4. Стадии и этапы разработки
а. Разработка ТЗ
б. Изучение предметной области
в. Разработка БД
в. Покупка хостинга
г. Изучение CSS и HTML
д. Разработка начальой страницы сайта
е. Разработка дополнительных страниц сайта
ж. Проведение тестирования 
-----5. Порядок контроля и приемки
5.1 Общие требования к приемке работы
На момент приемки на сайте должны находиться рецепты, распределенные по категориям и доступные для просмотра.
